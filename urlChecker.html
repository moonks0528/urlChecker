TCP 소켓 기반 URL 위험도 분석 시스템

사용자가 입력한 URL을 서버로 전송하고, 위험도를 분석하여 악성 사이트 접근을 사전에 차단하는 보안 시스템

프로젝트 개요

최근 피싱 사이트와 악성 URL을 이용한 개인정보 유출 피해가 증가하고 있다. 본 프로젝트는 사용자가 URL을 입력하거나 클릭했을 때, TCP 소켓 통신을 통해 서버로 URL을 전송하고 서버가 위험도를 분석한 뒤 결과를 클라이언트에게 반환하는 시스템이다.


이를 통해 사용자는 별도의 검사 과정 없이 URL의 안전성을 확인할 수 있으며, 위험한 사이트 접근을 사전에 차단할 수 있다.

주요 기능

URL 입력 및 전송

클라이언트에서 URL을 입력하고 서버로 검사 요청을 전송한다.

TCP 소켓 통신

Python Socket을 이용하여 클라이언트와 서버가 데이터를 주고받는다.

위험도 분석

URL 길이, 특수문자, 도메인 구조 등을 기준으로 위험 여부를 판단한다.

접속 제어

위험 URL로 판단되면 경고 메시지를 출력하고 접근을 제한한다.

적용 기술

  • Python Socket 기반 TCP 통신
  • 클라이언트-서버 구조 설계
  • URL 문자열 및 패턴 분석
  • SQLite 기반 악성 URL 데이터베이스
  • Scikit-learn 기반 머신러닝 분석 확장
  • 서버 연결 실패 및 응답 지연 예외 처리

시스템 동작 과정

1. URL 입력
2. 서버 전송
3. 위험도 분석
4. 결과 반환
5. 경고 또는 접속 허용
